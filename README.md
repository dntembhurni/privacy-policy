# Privacy Policy

**Bloomlabz Pvt Ltd**
**App:** Number Fun
**Effective date:** July 9, 2026

This Privacy Policy explains how Bloomlabz Pvt Ltd handles information in Number Fun.

Number Fun is designed as a child-directed number learning app with local-only progress, no accounts, no advertising, no analytics, no tracking, and no social features.

## Information stored on this device

Number Fun saves basic learning progress on the device so parents can review practice history and the app can continue from where the child left off.

This progress stays inside the app on the device and is not sent to Bloomlabz Pvt Ltd or another company.

## Information we do not collect

Number Fun does not ask for or collect:

* Child name
* Parent name
* Email address
* Phone number
* Photos or videos
* Voice recordings
* Location
* Contacts
* Calendar data
* Payment information
* Open-ended child content

Number Fun does not create user profiles or measure behavior for analytics or advertising.

## No ads, sale, or sharing

Number Fun does not include:

* Advertising
* Sponsored content
* Rewarded ads
* Subscriptions
* Paid upgrades
* In-app purchases

Bloomlabz Pvt Ltd does not sell, rent, or share child data with advertisers, data brokers, analytics providers, or social networks.

## Device text-to-speech

Number Fun uses the device text-to-speech feature to speak numbers and short learning prompts.

Number Fun does not:

* Record the child
* Use the microphone
* Upload speech audio

## Retention, deletion, and backups

Local progress remains on the device until:

* App is deleted, or
* Parent selects **Reset Progress** in Parent View

Reset Progress deletes saved Number Fun learning progress from that device.

Device backups, restores, family-sharing settings, and device transfer tools are controlled by the parent or device owner outside Number Fun.

## Children and parent control

Number Fun is intended for use by children with parent or guardian supervision.

Parent View is protected by a long press and includes:

* Progress review
* Reset controls
* Privacy Policy
* Terms and Conditions
* Data and Child Safety information
* Required notices

## Changes to this policy

Bloomlabz Pvt Ltd may update this Privacy Policy when Number Fun changes.

Updated policy will include a revised effective date.

## Contact

Parents can contact Bloomlabz Pvt Ltd through developer contact details provided on Number Fun app store listing.
